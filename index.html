How to play?!

Enter the any number and it will display the number and If it displays "Fizz" then the number you entered is a multiple of 3, or if it displays "Buzz" then it is a multiple of 5, lastly it will show "FizzBuzz" if the number is multiple of both.
There is a specific number on which it will display a special message!
